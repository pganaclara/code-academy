Adding a fallback font

When you start this exercise, the default font-family is "Roboto". But, we're importing the "Roboto" font using a Content Delivery Network (CDN). However, if the CDN is down or is loading too slowly, the "Roboto" font won't display correctly. We can add fallback font of "Courier" for the rare cases that the CDN isn't working properly.

If you want to see the fallback font in action after you've passed the checkpoint, delete 

 <link href="https://fonts.googleapis.com/css?family=Roboto" rel="stylesheet">
from the header element and run your code.
